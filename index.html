Research

Uncovering Sex Bias in Machine Learning Algorithms for Detecting Acute Myocardial Infarction using Electrocardiographic Data

This study investigates potential sex bias in ML models designed to detect Acute Myocardial Infarction (AMI) using electrocardiographic (ECG) data. Despite balancing male-to-female sample ratios, models consistently performed better for male patients, highlighting inherent biases in ECG pattern distributions. The findings emphasize the need for more nuanced strategies beyond dataset balancing to address disparities in healthcare AI.

ESM+: Modern Insights into Perspective on Text-to-SQL Evaluation in the Age of Large Language Models

This work evaluates the robustness of current metrics for Text-to-SQL tasks, focusing on large language models (LLMs). We identify limitations in existing metrics like Test Suite Execution Accuracy (EXE) and Exact Set Matching Accuracy (ESM) and propose ESM+, an improved metric with significantly lower false positive and negative rates. ESM+ provides a more reliable evaluation framework for Text-to-SQL models, and we release it as open-source for community use.

Emergency department triage prediction of clinical outcomes using machine learning models

This work explores the use of machine learning models to predict hospital and ICU admissions during emergency department triage. Using patient data from Emory Hospital, we developed separate predictive models for two critical outcomes: hospital admission and ICU admission. By analyzing clinical features such as vital signs, demographics, and visit details, our approach demonstrates the potential to enhance early risk identification and optimize resource allocation in emergency care.
